## 🚀 Projet phare — SmartOps

> **Plateforme interne modulaire de gestion des opérations** pour une entreprise de dispositifs médicaux.
> Un ERP maison qui couvre tout le cycle métier, du parc IT à la logistique chirurgicale.

<table>
  <tr>
    <td valign="top" width="50%">

**🏗️ Architecture**
- **Modulaire** : Parc IT, Livraison & Stocks, Importation, Commercial… (13+ modules sur la roadmap)
- **RBAC fin** + cloisonnement **multi-division**
- **API-first** (OpenAPI + JWT) pour alimenter un mobile unique
- **Workflow Designer** maison (éditeur canvas zoom/pan) = référentiel de la logique métier

</td>
    <td valign="top" width="50%">

**🧪 Du MVP à la cible**
- **Aujourd'hui** : PHP 8.2 · PDO · MySQL 8 — labo fonctionnel rapide
- **Cible** : microservices **NestJS · React · React Native · PostgreSQL · Redis · Keycloak**
- **Mobile** : React Native **offline-first** (SQLite + file de synchro)
- **Design system** custom (thème clair/sombre, composants partagés)

</td>
  </tr>
</table>
